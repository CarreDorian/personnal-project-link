# **__Voici la description des projets :__** 

######   
**Développement d'un algorithme quantique (Grover)**

- **Conception et implémentation** d'un algorithme **quantique** permettant une recherche optimisé dans une base de données non ordonnée. 
- La base de données contient des mots, impliquant une **vectorisation** (Bag Of Words) de celle-ci ainsi qu’une **présélection** par **dichotomie,** permettant de **réduire** le nombre de qubits nécessaires. 
- Création de **visualisations** des résultats.

###### **Réseau neuronal sans bibliothèque dédiée (My-first-IA)**

- **IA Prévisionnelle** prédisant des goûts cinématographiques en fonction d'un profil
- Le Programme peut également exécuter plusieurs IA en simultané grâce au 

       **multi-threading** ou au **multi-processing**, afin d'augmenter la

       fiabilité/rapidité d'obtention des résultats.

###### **Générateur de réseaux neuronaux en C++ (Deep-Learning-Generator)**

- permet de **créer** un réseau neuronal profond
- de **charger** les poids et en **déduire** le réseau neuronal correspondant
- de **sauvegarder** les poids.
- Une **interface de commande** (CLI) permet de l'utiliser.
